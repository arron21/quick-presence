Template Features

Exactly what You Need, Nothing more, Nothing less.

Fully Responsive

Responsive out of the box. The navigation automatically adapts to the amount of links you add to the main navigation.

Convenient Code

Clean and well written code guaranteed. This code is optimized to work in all major browsers and be extremely lightweight and efficient.

Parallax Images

Cool scrolling effect on background images throughout the page. You can add your own images to the img folder, and point to them in the <style> tag in the head of the index.html file.

Parallax Images

This background image can be changed in the <style> tag in the head of the index.html file.

Zoom Images

Clicking on these images enlarges them for easy viewing. Simply add data-action="zoom" to any <img> tag in your HTML and it will zoom.

Example: <img src="assets/img/new-zealand-lake-mountain-landscape-37650.jpg" data-action="zoom" />
